ALARM CLOCK :-
-----------
DESCRIPTION

Alarm Clock is a Python application that allows users to set alarms for specific times. When the alarm time is reached, the program plays a sound or displays a notification to alert the user. This project demonstrates the use of Python's date and time modules, GUI development, and event handling.

FEATURES 

Set alarms for specific hours and minutes.
Real-time clock display.
Alarm notification when the set time is reached.
Simple and user-friendly interface.
Easy to customize and extend.

PROJECT STRUCTURE :-
Alarm-Clock/
│
├── alarm_clock.py
├── README.md
└── assets/
    └── alarm_sound.wav (optional)
Example
Current Time: 08:15:30
Set Alarm:    08:16:00

Alarm Set Successfully!

When the clock reaches 08:16:00, the alarm will ring.

HOW IT WORKS

The program continuously checks the current system time.
It compares the current time with the user-defined alarm time.
Once both times match, the alarm is triggered
